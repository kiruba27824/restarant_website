# Great Indian Restaurant

A modern and responsive restaurant website developed to showcase authentic Indian cuisine, menu offerings, services, and contact information. The website provides an engaging user experience with a clean interface and mobile-friendly design.

## Live Demo

https://your-live-demo-link.vercel.app

## Features

- Responsive design for desktop, tablet, and mobile devices
- Modern and user-friendly interface
- Interactive navigation menu
- Home page with hero section
- About Us page
- Menu with food categories
- Featured dishes section
- Services section
- Customer testimonials
- Contact page with reservation/contact form
- Footer with social media links
